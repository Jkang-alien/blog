ion2cbioportal Rpackage

Last updated on Dec 3, 2019 1 min read Cancer, cbioportal, R

https://github.com/Jkang-alien/ion2cbioportal

This is Rpackage for making the cbioportal dataset from the ion torrent NGS result files. The formats of ion torrent result files are vcf and tsv. Those files convert to maf format and other data set for cbioportal. The package includes vignette. vignette("ion2cbioportal")
